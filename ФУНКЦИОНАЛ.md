# Функционал сайта АВТОМИР

## Реализованный функционал

### Каталог автомобилей
- **12 автомобилей** с полной информацией
- **Цены** отображаются для каждого автомобиля
- **Фильтры**: по марке, цене (от/до), году выпуска
- **Карточки автомобилей** с изображениями, ценами, характеристиками
- **Переход на детальную страницу** по клику на карточку

### Регистрация и Авторизация
- **Регистрация**:
  - Поля: Email, Username, Password, Confirm Password
  - Валидация паролей
  - Проверка на существующих пользователей
  - Сохранение в localStorage
  
- **Авторизация**:
  - Вход по email и паролю
  - Сохранение сессии пользователя
  - Отображение имени пользователя в шапке
  - Кнопка выхода

### Все страницы заполнены и работают:

1. **Главная** (`/`)
   - Hero-секция с призывом к действию
   - Карточки услуг
   - Карусель автомобилей
   - Гарантии (6 карточек)
   - Преимущества (4 карточки)
   - FAQ секция
   - Форма обратной связи

2. **Каталог** (`/catalog`)
   - 12 автомобилей с ценами
   - Фильтры по марке, цене, году
   - Карточки с изображениями и характеристиками

3. **Детальная страница автомобиля** (`/car/:id`)
   - Полная информация об автомобиле
   - Галерея изображений
   - Характеристики
   - Описание и особенности
   - Кнопки связи

4. **О нас** (`/about`)
   - Информация о компании
   - Фото команды
   - Преимущества работы
   - Кнопка "Оставить заявку"

5. **Как купить** (`/how-to-buy`)
   - 6 шагов процесса покупки
   - Подробное описание каждого этапа

6. **Преимущества** (`/advantages`)
   - 4 карточки с преимуществами
   - Подробные описания

7. **Отзывы** (`/reviews`)
   - 6 отзывов клиентов
   - Рейтинги (звезды)
   - Даты отзывов

8. **Контакты** (`/contact`)
   - Адрес, телефон, email
   - Время работы
   - Социальные сети
   - Карта (заглушка)
   - Ссылки на реквизиты и политику

9. **Регистрация** (`/register`)
   - Полностью рабочая форма
   - Валидация
   - Сохранение пользователей

10. **Авторизация** (`/login`)
    - Форма входа
    - Проверка данных
    - Сохранение сессии

### Навигация
- Все ссылки в меню работают
- Мобильное меню для маленьких экранов
- Активное состояние текущей страницы
- Плавные переходы между страницами

### Хранение данных
- Пользователи сохраняются в localStorage
- Сессия пользователя сохраняется
- Данные сохраняются между перезагрузками страницы

## Как использовать

1. **Запустите проект**: `npm run dev`
2. **Откройте**: http://localhost:5173
3. **Попробуйте**:
   - Просмотреть каталог с машинами и ценами
   - Зарегистрироваться (создастся аккаунт)
   - Войти (отобразится имя в шапке)
   - Перейти по всем вкладкам меню
   - Использовать фильтры в каталоге

## Адаптивность
- Все страницы адаптированы для мобильных устройств
- Мобильное меню работает корректно
- Карточки и сетки адаптируются под размер экрана
