# Инструкция по запуску проекта

## Проблема с PowerShell

Если вы видите ошибку о политике выполнения PowerShell, используйте один из способов ниже.

## Способ 1: Использовать CMD (Command Prompt)

1. Откройте **Command Prompt** (cmd) вместо PowerShell
2. Перейдите в папку проекта:
   ```
   cd "C:\Users\rapto\OneDrive\Рабочий стол\автосалон вуэ"
   ```
3. Запустите установку:
   ```
   npm install
   ```
4. Запустите проект:
   ```
   npm run dev
   ```

## Способ 2: Изменить политику PowerShell

Откройте PowerShell **от имени администратора** и выполните:

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

Затем в обычном PowerShell выполните:
```powershell
npm install
npm run dev
```

## Способ 3: Использовать готовые bat-файлы

В папке проекта есть файлы:
- `install.bat` - для установки зависимостей
- `start.bat` - для запуска проекта

Просто дважды кликните на них.

## После установки

После успешной установки зависимостей, запустите проект:

```bash
npm run dev
```

Проект будет доступен по адресу: **http://localhost:5173**
