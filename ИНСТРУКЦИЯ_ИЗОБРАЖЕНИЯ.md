# Инструкция по добавлению изображений автомобилей

## Структура папок

Создайте следующую структуру папок в проекте:

```
public/
  images/
    cars/          # Изображения автомобилей
    hero/          # Изображение для главной страницы
    about/         # Изображение для страницы "О нас"
    icons/         # Значки для карточек "С пробегом", "Новые", "Под ваш бюджет"
    logo/          # Логотипы компании (шапка, футер, favicon)
```

## Как добавить изображения

### 1. Изображения автомобилей

Поместите изображения автомобилей в папку `public/images/cars/`:

- `toyota-rav4-2022.jpg` - главное изображение Toyota RAV4
- `toyota-rav4-2022-1.jpg` - дополнительное изображение 1
- `toyota-rav4-2022-2.jpg` - дополнительное изображение 2
- `toyota-rav4-2022-3.jpg` - дополнительное изображение 3
- `audi-q5-2021.jpg` - главное изображение Audi Q5
- `honda-cr-v-2021.jpg` - главное изображение Honda CR-V
- и т.д.

### 1.1. Значки для карточек услуг

Чтобы заменить эмодзи на реальные значки в блоке "С пробегом / Новые / Под ваш бюджет" на главной странице (`src/views/Home.vue`), сложите иконки в папку `public/images/icons/` со следующими именами:

- `service-used.svg` (или .png/.webp) — для карточки **«С пробегом»**
- `service-new.svg` — для карточки **«Новые»**
- `service-budget.svg` — для карточки **«Под ваш бюджет»**

После загрузки файлов обновите разметку карточек услуг, чтобы использовать `<img src="/images/icons/service-used.svg">` и т.п. вместо эмодзи.

### 2. Изображение для главной страницы

Поместите изображение автомобиля для hero-секции:
- `public/images/hero/hero-car.jpg`

### 3. Изображение для страницы "О нас"

Поместите фото команды:
- `public/images/about/team.jpg`

### 4. Логотипы

- Основной логотип сайта: `public/images/logo/site-logo.png`
- Дополнительная версия (на тёмном фоне): `public/images/logo/site-logo-invert.png`
- Если нужен favicon, добавьте `public/images/logo/favicon.png` и подключите в `index.html`

## Форматы изображений

Рекомендуемые форматы:
- **JPG** - для фотографий (меньший размер)
- **PNG** - для изображений с прозрачностью
- **WebP** - современный формат (лучшее сжатие)

## Рекомендуемые размеры

- **Автомобили в каталоге**: 600x400px или 800x600px
- **Hero изображение**: 1200x800px или больше
- **Изображение команды**: 800x600px
- **Галерея автомобиля**: 800x600px каждое

## Где найти изображения

1. **Бесплатные стоки:**
   - Unsplash (https://unsplash.com) - поиск "car", "automobile"
   - Pexels (https://www.pexels.com) - поиск "car"
   - Pixabay (https://pixabay.com) - поиск "car"

2. **Специализированные сайты:**
   - Автомобильные каталоги
   - Официальные сайты производителей

3. **Собственные фотографии:**
   - Сфотографируйте автомобили сами
   - Используйте профессиональные фото

## Важно

- Все изображения должны быть в папке `public/images/`
- Имена файлов должны совпадать с указанными в коде
- Если изображение не найдено, будет показан placeholder
- Оптимизируйте изображения перед загрузкой (используйте инструменты сжатия)

## Проверка

После добавления изображений:
1. Убедитесь, что файлы находятся в правильных папках
2. Перезапустите сервер разработки: `npm run dev`
3. Проверьте, что изображения отображаются на сайте
