# Автосалон - Vue.js сайт

Многостраничный сайт автосалона, созданный на Vue.js 3 с использованием Vue Router.

## Возможности

- Главная страница с популярными автомобилями
- Каталог автомобилей с фильтрацией
- Детальная страница автомобиля
- Страница "О нас"
- Страница контактов с формой обратной связи
- Адаптивный дизайн для всех устройств

## Технологии

- Vue.js 3
- Vue Router 4
- Vite
- CSS3 (адаптивный дизайн)

## Быстрый старт

1. Установите зависимости:
```bash
npm install
```
2. Запустите проект в режиме разработки:
```bash
npm run dev
```
   По умолчанию приложение поднимается на `http://localhost:5173`.

Если нужно проверить сборку без разработки, можно собрать проект и открыть превью:
```bash
npm run build
npm run preview
```
Превью также будет доступно на 5173 порту.

## Сборка

Для создания production сборки:

```bash
npm run build
```

## Структура проекта

```
├── src/
│   ├── components/      # Компоненты (Header, Footer)
│   ├── views/           # Страницы (Home, Catalog, CarDetail, About, Contact)
│   ├── layouts/         # Layout компоненты
│   ├── router/          # Конфигурация роутера
│   ├── App.vue          # Главный компонент
│   ├── main.js          # Точка входа
│   └── style.css        # Глобальные стили
├── index.html
├── package.json
└── vite.config.js
```

## Страницы

- `/` - Главная страница
- `/catalog` - Каталог автомобилей
- `/car/:id` - Детальная страница автомобиля
- `/about` - О нас
- `/contact` - Контакты
