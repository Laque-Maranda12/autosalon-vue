# Иконки карточек услуг

Добавьте сюда реальные иконки для карточек на главной странице.

Имена файлов, которые сейчас используются в `src/views/Home.vue`:
- `service-used.svg` — иконка карточки **«С пробегом»** (`/images/icons/service-used.svg`).
- `service-new.svg` — иконка карточки **«Новые»** (`/images/icons/service-new.svg`).
- `service-budget.svg` — иконка карточки **«Под ваш бюджет»** (`/images/icons/service-budget.svg`).

Если заменяете иконку, обновите соответствующий файл, **сохранив имя**. Если добавляете новый файл с другим именем, не забудьте поменять путь в `src/views/Home.vue`.
