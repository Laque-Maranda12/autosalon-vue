# Изображения автомобилей

## Да, изображения машин будут работать!

Сейчас в коде настроена поддержка реальных изображений. 

## Где размещать изображения

Создайте папки в проекте:
```
public/
  images/
    cars/          ← Сюда поместите фото автомобилей
    hero/          ← Главное изображение для главной страницы
    about/         ← Фото команды для страницы "О нас"
```

## Как добавить изображения

1. **Создайте папки** (вручную или через файловый менеджер):
   - `public/images/cars/`
   - `public/images/hero/`
   - `public/images/about/`

2. **Поместите изображения** с правильными именами:
   - `toyota-rav4-2022.jpg` - для Toyota RAV4
   - `audi-q5-2021.jpg` - для Audi Q5
   - `honda-cr-v-2021.jpg` - для Honda CR-V
   - `hero-car.jpg` - для главной страницы
   - `team.jpg` - для страницы "О нас"

3. **Перезапустите сервер**: `npm run dev`

## Что происходит сейчас

- Если изображение найдено → показывается реальное фото
- Если изображение не найдено → показывается placeholder (временная заглушка)

## Где найти изображения

- **Unsplash**: https://unsplash.com/s/photos/car
- **Pexels**: https://www.pexels.com/search/car/
- **Pixabay**: https://pixabay.com/images/search/car/

Или используйте свои фотографии автомобилей!

## Технические детали

Все изображения должны быть в папке `public/`, чтобы они были доступны по пути `/images/...`

Подробная инструкция в файле `ИНСТРУКЦИЯ_ИЗОБРАЖЕНИЯ.md`
